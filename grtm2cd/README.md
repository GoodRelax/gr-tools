### The Story

Sometimes we find a treasure map and want to keep it secret.  
Why don't you split the map in two, and let a cat and a dog each keep a piece?  
When we want to use the map, we just call both the cat and the dog.

This web tool helps in situations like that.  
Don't worryâ€”this page works entirely on your computer, and your data is never sent to the web.

### What is this?

GRTM2CD is a completely private, browser-based steganography tool.  
It takes any secret file (your "Treasure Map") and securely hides it  
by splitting it across two ordinary images (like a "Cat" and a "Dog").  
You need both images to recover the secret.

### How to Use

1. Open `index.html` in your browser (no installation or server needed!).
2. **Encode:** Drop your secret file, then drop two images (for example, a Cat and a Dog).
3. Download the two new generated images.
4. **Decode:** Drop the two generated images back into the tool to recover your exact secret file.
