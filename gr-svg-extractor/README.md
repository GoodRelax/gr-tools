# GR SVG Extractor

A lightweight, single-file web application to extract clean SVGs from clipboard data of PowerPoint or Excel.

1.  Open `gr-svg-extractor.html` in **Chrome** or **Edge**.
2.  **Copy** (`Ctrl+C`) shapes or charts in PowerPoint or Excel.
3.  Click **Paste** in this tool.
4.  **Download** the `.svg` file or copy the code.
